# OFFICE（浏览器单文件版）

一款在浏览器里运行的叙事向办公室题材小游戏：**单 HTML、无构建、无后端**。双击或用浏览器打开即可玩；

---

## 操作说明（游戏内）

- **移动**：方向键或 `WASD`（可按住连走）
- **交互 / 确认**：`Z`、`Enter`
- **取消**：`X`、`Esc`
- **日程**：`Tab`
- **地图**：`M`
- **主菜单新增**：`个人叙事` 可浏览跨周目记忆碎片拼出的个人故事。

具体以游戏内底部提示为准。

---

## 存档说明

- 使用浏览器 **`localStorage`** 保存进度与跨周目元数据（黑话字典、成就、结局、个人叙事记忆碎片、周目 meta 选择等）。
- 换浏览器、清站点数据、隐私模式可能导致存档不可用。
- 正式发布给他人时，可在 README 里提醒：**用在线 https 链接打开存档更稳定**。

## 反馈与修改

这是个人向 demo，剧情与梗均为虚构。若要改文案、数值或加功能，直接编辑 HTML 内 `<script>` 与 `<style>` 即可；无需安装 Node 或打包工具。
